COURSE: CS501 - Machine Learning
DEPARTMENT: Computer Science
SEMESTER: 5th Semester
CREDITS: 4

COURSE DESCRIPTION:
An introduction to machine learning covering supervised, unsupervised, and reinforcement learning. Focus on both theoretical foundations and practical implementation using Python and popular ML frameworks.

COURSE OBJECTIVES:
1. Understand the mathematical foundations of machine learning
2. Implement and evaluate supervised learning algorithms
3. Apply unsupervised learning for clustering and dimensionality reduction
4. Build neural networks and deep learning models
5. Apply ML to real-world datasets

UNIT 1: Foundations
- Linear algebra review (vectors, matrices, eigenvalues)
- Probability and statistics review
- Optimization: gradient descent, stochastic gradient descent
- Python scientific stack: NumPy, Pandas, Matplotlib
Recommended Books: Pattern Recognition and Machine Learning (Bishop), Hands-On Machine Learning (Géron)

UNIT 2: Supervised Learning
- Linear regression and polynomial regression
- Logistic regression and classification
- Support Vector Machines (SVM)
- Decision trees and random forests
- Model evaluation: cross-validation, bias-variance tradeoff
Recommended Books: Hands-On Machine Learning (Géron), Pattern Recognition and Machine Learning (Bishop)

UNIT 3: Neural Networks and Deep Learning
- Perceptrons and multi-layer networks
- Backpropagation algorithm
- Activation functions, regularization, dropout
- Convolutional Neural Networks (CNN)
- Recurrent Neural Networks (RNN) and LSTM
Recommended Books: Deep Learning (Goodfellow), Hands-On Machine Learning (Géron)

UNIT 4: Unsupervised Learning
- K-means clustering
- Hierarchical clustering
- Principal Component Analysis (PCA)
- Autoencoders
Recommended Books: Pattern Recognition and Machine Learning (Bishop)

UNIT 5: Advanced Topics
- Transformer architecture and attention mechanisms
- Transfer learning and fine-tuning
- Introduction to reinforcement learning
- Ethical AI and fairness in ML
Recommended Books: Artificial Intelligence: A Modern Approach (Russell), Reinforcement Learning (Sutton)

UNIT 6: Project and Applications
- End-to-end ML pipeline
- Feature engineering and selection
- Deployment with Flask/FastAPI
- NLP applications with transformers
Recommended Books: Data Science from Scratch (Grus), Natural Language Processing with Python (Bird)

PRACTICAL/LAB: Python implementations; Kaggle projects; group project
ASSESSMENT: 4 assignments (40%), Mid-term (20%), Final Project (40%)
