COURSE: CS301 - Data Structures and Algorithms
DEPARTMENT: Computer Science
SEMESTER: 3rd Semester
CREDITS: 4

COURSE DESCRIPTION:
This course covers fundamental data structures and algorithm design techniques. Students will learn to analyze algorithm complexity using Big-O notation and implement efficient solutions to computational problems.

COURSE OBJECTIVES:
1. Understand and implement fundamental data structures
2. Analyze time and space complexity of algorithms
3. Apply algorithm design paradigms: divide and conquer, dynamic programming, greedy
4. Solve graph problems using BFS, DFS, shortest path algorithms
5. Implement searching and sorting algorithms

UNIT 1: Algorithm Analysis
- Big-O, Omega, Theta notation
- Recurrence relations and Master theorem
- Space complexity analysis
Recommended Books: Introduction to Algorithms (Cormen), Data Structures and Algorithm Analysis in C++ (Weiss)

UNIT 2: Linear Data Structures
- Arrays and linked lists (singly, doubly, circular)
- Stacks: implementation and applications
- Queues: circular queue, deque, priority queue
Recommended Books: Introduction to Algorithms (Cormen)

UNIT 3: Non-Linear Data Structures
- Trees: binary tree, BST, AVL, Red-Black tree
- Heaps and heap sort
- Trie and B-trees
Recommended Books: Introduction to Algorithms (Cormen)

UNIT 4: Graphs
- Graph representations: adjacency matrix, adjacency list
- BFS and DFS traversals
- Shortest path: Dijkstra, Bellman-Ford
- Minimum spanning tree: Kruskal, Prim
Recommended Books: Introduction to Algorithms (Cormen), Discrete Mathematics and Its Applications (Rosen)

UNIT 5: Sorting and Searching
- Comparison sorts: quicksort, mergesort, heapsort
- Linear sorts: counting sort, radix sort, bucket sort
- Binary search and variants
Recommended Books: Introduction to Algorithms (Cormen), Data Structures and Algorithm Analysis in C++ (Weiss)

UNIT 6: Algorithm Design Paradigms
- Divide and conquer
- Dynamic programming: memoization, tabulation
- Greedy algorithms
- Backtracking
Recommended Books: Introduction to Algorithms (Cormen), Algorithm Design (Kleinberg)

PRACTICAL/LAB: Implementation in C++ or Python; weekly assignments
ASSESSMENT: 3 assignments (30%), Mid-term (30%), End-term (40%)
